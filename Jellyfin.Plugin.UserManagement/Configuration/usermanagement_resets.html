Password Resets

When a user starts the Forgot Password flow, Jellyfin writes a reset code to a file on the server. This tab surfaces those codes so you can pass one to the user without needing file access to the server.
It is not recommended to use reset code extraction over a non-HTTPS connection. Reset codes are sent over this connection so any non-encrypted traffic can be surveilled.
